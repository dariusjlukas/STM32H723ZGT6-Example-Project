# STM32H723ZGT6 Example Project

An example C++/Makefile project for the STM32H723ZGT6 which demonstrates how to use GPIO, USB CDC, and the debugger.

## Dependencies

- ARM-GNU toolchain (update `gcc-path` in Makefile).
- Make
- OpenOCD
- dfu-util
- stlink
- STM32CubeMX (for changing configuration)
- VSCode *Cortex-Debug* plugin
- VSCode *Makefile Tools* plugin

## Build

```bash
make
```

## Flash (ST-Link)

```bash
make flash
```

## Flash (USB DFU mode)

```bash
make flash_dfu
```

## Debug (VSCode)

Run and Debug -> Start Debugging (Debug (OpenOCD))
